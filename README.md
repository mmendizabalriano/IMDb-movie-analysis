# IMDb Movie Data Analysis

This project performs an exploratory data analysis (EDA) on a dataset of IMDb movies using Python.

The goal of the analysis is to explore patterns in movie ratings, genres, production trends, and director performance. The project includes data cleaning, transformation, and visualization using pandas and matplotlib.

## Questions explored

- How are movie ratings distributed?
- How have movie ratings evolved over time?
- Which genres are most common?
- Which genres receive the highest ratings?
- Do longer movies receive better ratings?
- Is there a relationship between budget and rating?
- Which directors consistently produce highly rated movies?

## Tools

- Python
- pandas
- matplotlib
- numpy
